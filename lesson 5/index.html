:active - Применяется при активации пользователем элемента. :link - Применяется к ссылкам, на которые пользователь ещё не нажимал. :visited — Применяется к ссылкам, на которые пользователь нажимал. :focus - Применяется к элементу при получении им фокуса. :hover — срабатывает, когда пользователь наводит на элемент мышью. :nth-child - дит один или более элементов, основываясь на их позиции среди группы соседних элементов. :last-child - находит последний элемент в родителе. :first-child - находит первый элемент в родителе. :root - находит корневой элемент дерева документа. :enabled - находит любой включенный элемент
